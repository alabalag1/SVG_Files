\doxysubsection{Detailed Description}
A vector of pointers to \mbox{\hyperlink{class_figure}{Figure}} objects was used to store all the figures that are created or read and later saved in a file. If a normal \mbox{\hyperlink{class_figure}{Figure}} object was used instead of a pointer to \mbox{\hyperlink{class_figure}{Figure}} object for the vector, there would be object slicing.
