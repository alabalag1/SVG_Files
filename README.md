# SVG_Files

Parsing SVG files to objects in C++, giving the ability to easily modify them.  

## Features

* Open ingfiles  
* Closing files  
* Save and Save As files  
* Printing all Figures  
* Creating a new Figure  
* Erasing a Figure  
* Translating figures horizontally and vertically
